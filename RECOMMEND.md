- 快速将json串转换为java bean的AS插件：`GsonFormat`
- 资源文件id快速生成插件：`Android layout id converter`
- [JVM内存调试插件](https://zhuanlan.zhihu.com/p/25110433?utm_medium=email&utm_source=gank.io)：`JVM Debugger Memory View`
